# niri

Niri doesn't have a package for Debian, so compile necessary packages for Debian 13 stable.

Files uploaded to the repository.

✅ Install:

`dpkg -i niri_26.4.0-1_amd64.deb`


Niri to work correctly depends on Xwayland-satellite, you will also need to install it.

`dpkg -i xwayland-satellite_0.8.1-1_amd64.deb`
